## Приоритет реализации (быстрый ROI)

1. **Постоянные URL + Schema.org** → SEO-трафик за 1–2 недели
2. **Текстовый поиск** → одна строка input + JS-фильтр
3. **Избранное** → localStorage + иконка сердца на карточке
4. **Ипотечный калькулятор** → виджет на странице объекта
5. **Реальная мультиязычность** → JSON-файлы переводов + switcher
