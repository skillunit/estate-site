## 🔴 Критично (9) — прямо влияет на конверсию

| # | Функция | Описание |
|---|---------|----------|
| 1 | Ипотечный калькулятор | Нет расчёта ежемесячного платежа прямо в карточке |
| 2 | Реальная мультиязычность | EN/HE в UI есть, контент не переключается |
| 3 | Fullscreen-карта | Нет режима «только карта» с боковой панелью объектов |
| 4 | Сравнение объектов | Нельзя выбрать 2–4 объекта и сравнить по таблице |
| 5 | Онлайн-запись к агенту | Нет бронирования конкретного слота звонка / встречи |
| 6 | Страница застройщика | Нет страниц застройщиков с портфолио и репутацией |
| 7 | График динамики цен | Нет исторического графика роста по объекту / локации |
| 8 | Schema.org (JSON-LD) | Нет RealEstateListing — объекты не в расширенных результатах Google |
| 9 | Постоянные URL объектов | SPA с showPage() — нельзя поделиться прямой ссылкой, нет индексации (в projects.html частично решено через `?prop=`) |

4. **Fullscreen-карта** → режим «только карта» с боковой панелью
